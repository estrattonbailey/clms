# clms
Command-line meta scraper for node.

<img src='https://raw.githubusercontent.com/estrattonbailey/clms/master/demo.gif' style='max-width: 600px; margin: 1em 0;' />

## Install
```bash
npm i clms -g
```

# Usage
```bash
clms <url>
```
